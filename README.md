# Landing Page - Primitive Man (Rock Band)

This is a promotional web page designed for the Sludge/Doom metal band **Primitive Man**. The project's goal is to provide a platform where fans can check upcoming concert dates, explore the band's discography, view an event gallery, and connect via social media.

## 🚀 Features

- **Responsive Design (Mobile First):** The page automatically adapts to mobile devices, tablets, and desktop computers using Media Queries.
- **Interactive Navigation:** Includes a fully functional "hamburger" menu to enhance the user experience on mobile devices.
- **Detailed Sections:**
    - **Hero Section:** Striking visual presentation featuring the band's style.
    - **Concert Table:** Organized information displaying dates, cities, and schedules.
    - **Discography:** A visual list of featured albums.
    - **Gallery:** An image grid optimized using CSS Grid.
- **Dark/Metal Aesthetics:** The interface is designed with a dark color palette (#1a1a1a) and yellow accents (#ffcc00) to align with the band's visual identity.

## 🛠️ Technologies Used

The following tools and languages were used in this project:

- **HTML5:** Full semantic structure (`<header>`, `<main>`, `<section>`, `<footer>`).
- **CSS3:** - **Flexbox:** For element alignment and distribution.
    - **CSS Grid:** For the symmetrical layout of the image gallery.
    - **Media Queries:** To ensure a fully responsive design.
    - **Advanced Properties:** Implementation of `object-fit: cover` to prevent image distortion.
- **JavaScript (Vanilla JS):** Logic for the mobile dropdown menu functionality.
